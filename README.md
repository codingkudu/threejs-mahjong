# 發 · Green Dragon Mahjong Tile

An interactive 3D mahjong tile featuring the Chinese character 發 (fā) — meaning Prosperity, Fortune, and To Flourish.

Built with [Three.js](https://threejs.org/).
Really built with AI. Something to demostrate AI to my nephew with.

## Features

- Rotating 3D mahjong tile with realistic ivory and green materials
- Hand-etched character with bump mapping for carved depth
- Drag to rotate, scroll to zoom
- Decorative crosshatch back pattern
- Responsive design with mobile touch support

## Run Locally

Open `index.html` in your browser. No build step or server needed.
